1. Overview

Mind Influence Lab is a privacy-first research tool designed for individuals interested in exploring consciousness-matter interaction through controlled experiments. This privacy policy explains what data is collected, how it is used, and your rights as a user.

2. Data Collection

2.1 What We Collect

The following data is collected and stored locally on your device:

  • Session Data: Experiment results including RNG deviation, sensor readings, and environmental metrics
  • Profile Data: XP, levels, streaks, and achievement progress
  • Environmental Data: Noise levels, magnetic field, pressure, gyroscope, WiFi signal strength, Bluetooth signal strength
  • App Usage: Session timestamps and duration

2.2 What We Do NOT Collect

The following data is NOT collected:

  • Personal Identifiers: No name, email, or phone number
  • Physical Location: No GPS coordinates or location history are recorded
  • Contact Information: No contacts or call logs
  • Browsing History: No browsing data
  • Device Identifiers: No IMEI or advertising ID
  • Audio Recordings: Microphone measures decibel levels only, no audio is recorded
  • Network Content: WiFi and Bluetooth scanning collects signal strength only, no data content is accessed
  • User Data Transmission: No personal or experiment data is sent to external servers

3. Sensor Permissions

3.1 Microphone (Optional)

Permission: RECORD_AUDIO

Purpose: Measure ambient noise levels in decibels for environmental context during experiment sessions

What Is Stored: Numeric decibel values only

What Is NOT Stored: No audio recordings, no speech content, no sound files

Processing: Audio stream is processed in real-time for amplitude calculation only. All audio data is immediately discarded after decibel value is extracted

Optional: Yes. This feature can be disabled in settings. The app functions fully without this permission

3.2 Storage Permissions

Permission: READ_EXTERNAL_STORAGE and WRITE_EXTERNAL_STORAGE

Purpose: Save session data and export or import CSV files for research analysis

What Is Accessed: App-specific folder only

What Is NOT Accessed: User personal files, photos, music, or other app data

Optional: No. Required for core functionality including data export

3.3 Location Permission (Required for WiFi/Bluetooth)

Permission: ACCESS_FINE_LOCATION or ACCESS_COARSE_LOCATION

Purpose: Android operating system requires location permission to access WiFi and Bluetooth hardware identifiers for signal strength scanning

What Is Stored: Signal strength values (RSSI) only

What Is NOT Stored: No GPS coordinates, no physical location history, no location tracking

Clarification: This permission is required by Android to scan for nearby signals, but the app does not use it to determine or store your physical location

Optional: No. Required for Advanced Lab environmental scanning features

3.4 Other Sensors (No Permissions Required)

The following sensors require no permissions and are used for environmental context:

  • Magnetometer: Environmental magnetic field measurements in microtesla
  • Accelerometer: Movement detection for motion quality scoring
  • Gyroscope: Rotation tracking for orientation data
  • Barometer: Atmospheric pressure measurements in hectopascals

4. Data Storage

  • Storage Location: Android internal storage in the app-specific directory
  • Encryption: Android sandbox protection for app-specific storage
  • Backup: User-controlled through manual CSV export
  • Deletion: Automatic when app is uninstalled
  • Access: App only. No other apps can access this data

5. Data Transmission and Internet Access

  • Is data sent to servers: No personal or experiment data is sent
  • Is data shared with third parties: No
  • Does app require internet: No, except for optional Quantum RNG (QRNG) feature
  • QRNG Internet Usage: If enabled in Advanced Lab, the app requests random numbers from a third-party QRNG API. No user data is included in these requests.
  • Are there analytics SDKs: No
  • Are there advertising SDKs: No

6. User Rights

  • Delete Your Data: Uninstall the app. All data is deleted automatically
  • Disable Sensors: Use Settings to toggle individual sensors off
  • No Account Required: No login means no account deletion is needed

2. 数据收集

我们收集的数据

  • 会话数据：实验结果，包括 RNG 偏差、传感器读数和环境指标
  • 个人资料数据：经验值、等级、连续天数和成就进度
  • 环境数据：噪声水平、磁场、压力、陀螺仪、WiFi 信号强度、蓝牙信号强度
  • 应用使用数据：会话时间戳和持续时间

我们不收集的数据

  • 个人标识符：无姓名、电子邮件或电话号码
  • 物理位置：无 GPS 坐标或位置历史记录
  • 联系信息：无联系人或通话记录
  • 浏览历史：无浏览数据
  • 设备标识符：无 IMEI 或广告 ID
  • 音频录音：麦克风仅测量分贝水平，不录制音频
  • 网络内容：WiFi 和蓝牙扫描仅收集信号强度，不访问数据内容
  • 用户数据传输：无个人或实验数据发送到外部服务器

麦克风（可选）

权限：RECORD_AUDIO

目的：测量实验会话期间的环境噪声水平（分贝）以提供环境背景

存储内容：仅数字分贝值

不存储内容：无音频录音、无语音内容、无声音文件

存储权限

权限：READ_EXTERNAL_STORAGE 和 WRITE_EXTERNAL_STORAGE

目的：保存会话数据并导出或导入 CSV 文件以进行研究分析

位置权限（WiFi/蓝牙所需）

目的：Android 操作系统要求位置权限才能访问 WiFi 和蓝牙硬件标识符以进行信号强度扫描

存储内容：仅信号强度值 (RSSI)

不存储内容：无 GPS 坐标、无物理位置历史、无位置跟踪

说明：此权限是 Android 扫描附近信号所必需的，但本应用不使用它来确定或存储您的物理位置
